# String Concatenation

- We can combine strings with a plus sign
```
console.log("a" + "b");

// result: "ab"
```
```
console.log("a" + " "+ "b");

// result: "a b"
```
- Challenge: create an alert that combines, or concatenates the message and name into a single message:
```
let message = "Hello";
let name = "Grace";
alert(message + " there, " + name);
```
